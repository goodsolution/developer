Procedure for course returning

(Remember, return only courses which was bought in right term)
1. Customer contact with us by email and say, that he want to return course
2. Find this order in our system and click details
3. Click in return button for this purchased course
4. Search this order in payu panel, and return the same amount as price of course.
5. Make correction invoice for order in wFirma.
6. Upload correction invoice to this order.
7. Notify customer by email about new invoice and return performing.
