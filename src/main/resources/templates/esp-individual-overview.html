Kurs Indywidaulny


Jaki jest zakres kursu?
  • Java
  • SQL
  • Spring Boot
  • Bazy danych na przykładzie MySQL
  • Elementy frontu: HTML, CSS, Javascript, JQuery, Ajax
  • Ze względu na specyfikę kursu indywidualnego zakres można dostosować do potrzeb kursanta.
  • Możliwość przygotowania pod kątem pracy jako programista lub tester

Jak to wygląda?
  • Spotkania 2h zegarowe co tydzień lub co 2 tygodnie lub raz na miesiąc (do wyboru)
  • Zajęcia online za pomocą komunikatora z możliwością udostępniania pulpitu np. Skype
  • Godziny ustalamy indywidualnie
  • Nastawienie na praktykę, kodujemy od pierwszych zajęć
  • Najpierw piszemy małe programiki, wraz z poszerzaniem wiedzy większe, a pod koniec kursu piszemy dużą aplikację
  • Po każdych zajęciach praca domowa a podczas lekcji podsyłam niezbędne materiały
  • Code review pisanych programów
  • Nauka projektowania architektury aplikacji
  • Nauka projektowania i tworzenia API REST
  • Poznawanie popularnych formatów danych jak np. JSON, XML, YAML
  • Przygotowanie do rozmów o pracę
  • Rozliczenia za każdą godzinę. Płatność po każdych zajęciach
  • W każdym momencie możesz przerwać kurs po wcześniejszym poinformowaniu
  • Jak dotąd 100% kursantów, którzy uczestniczyli w kursie znalazło pracę w IT
